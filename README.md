# Razer_GRUB_Theme
GRUB Theme for Razer Fans — For Gamers. By Gamers.

![preview.png](preview.png)

## Getting Started

```bash
git clone https://github.com/CasearF/Razer_GRUB_Theme.git
cd Razer_GRUB_Theme
chmod +x install.sh
sudo ./install.sh
```
